# **FlowScript Pricing Model (Final Draft)**

## **1. **FlowScript Free

A fully usable trial tier designed to let people *experience* FlowScript without giving away the full product.

### **What Free Users Get**

- **Full editor access (with limitations)**   They can open the editor, create nodes, connect flows, and explore the tool.
- **Limited node types (future constraint; not enforced at 1.0)**   At launch, all built‑in nodes are available because the library is small. Later, when the library grows, Free can be capped to a subset.
- **Limited projects →** ***No saving***   This is the strongest and cleanest limitation. Free users can experiment, but they can’t accumulate work. It creates a natural upgrade moment without crippling the experience.
- **DSL View: Read‑Only**   Perfect. They can *see* the DSL, understand the value, but not edit or export it. This is a powerful upsell without blocking learning.
- **No custom nodes**   Not part of 1.0 anyway, but keeping it listed future‑proofs the tier.
- **No export formats**   They can’t turn their work into JSON, .flowfile, .frame, or text. Again, a natural upgrade moment.
- **No cloud sync**
- **No collaboration**
- **No version history**  

------

## **2. FlowScript Desktop — One‑Time Purchase (Per Major Version)**

A full, offline, local‑first version of FlowScript that you own forever.

### **What you get**

- Full editor
- Unlimited projects
- Unlimited node types
- DSL view
- Custom nodes
- Templates
- Export formats
- Local version history
- Local debugging tools
- Plugins/extensions (if added later)
- Everything except cloud/team features

### **Versioning**

- Buy **FlowScript 1.x** → get all updates from **1.0 → 1.9.x**
- When **FlowScript 2.0** releases → optional upgrade
- If you stay on 1.x → it keeps working forever
- No forced upgrade
- No feature removals
- No punishment for sticking to an older version

### **Upgrade pricing**

- Discounted upgrade from 1.x → 2.x
- Discount amount TBD (e.g., 30–50% off)

------

## **3. FlowScript Pro — Subscription**

A cloud service layer for professionals, teams, and businesses.

### **What you get**

- Cloud sync
- Cloud backups
- Real‑time collaboration
- Shared projects
- Team permissions
- Cloud version history
- Cloud templates marketplace
- Integrations (GitHub, Supabase, etc.)
- Priority updates
- Pro‑only features as they evolve

### **Version‑agnostic**

- Works with any major version of FlowScript Desktop
- You are never forced to upgrade your Desktop version to use Pro

------

## **4. FlowScript Desktop Owner → Pro Upgrade**

A discounted subscription for users who already bought the Desktop version.

### **Why**

- You already paid for the editor
- You only pay for the cloud
- You are rewarded for supporting FlowScript early

### **Example pricing**

- FlowScript Pro: $12/month
- Desktop Owner Upgrade: $8/month

(Exact numbers TBD)

------

## **5. FlowScript Runtime (Included with Frame Walker)**

A restricted, embedded version of FlowScript for game players.

### **What it includes**

- Node editor for game logic
- Modding tools
- Automation tools

### **What it does NOT include**

- DSL view
- Custom nodes
- Cloud sync
- Export formats
- Templates
- Collaboration
- Version history

### **Purpose**

- Enables modding and mech logic editing
- Does not replace FlowScript Desktop or Pro
- Avoids double‑charging
- Avoids cannibalizing FlowScript as a standalone tool

------

# **Summary**

FlowScript is a **tool first**, with:

- a **free tier** for exploration
- a **one‑time purchase** for ownership
- a **subscription** for cloud/team features
- a **runtime** for Frame Walker

This model is:

- fair
- sustainable
- indie‑friendly
- respectful
- future‑proof
- aligned with your values and your audience
